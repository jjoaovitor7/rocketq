## :rocket: Tecnologias utilizadas

- CSS

- NodeJS
- EJS
- Sqlite3 (Banco de Dados)

---

## :computer: Como baixar o projeto e iniciar o projeto

```bash
  // Clonar o repositório
  $ git clone https://github.com/jjoaovitor7/rocketq

  // Entrar no diretório
  $ cd rocketq

  // Criar o Banco de Dados
  $ node src/db/init.js

  // Iniciar o servidor
  $ node src/server.js
```
